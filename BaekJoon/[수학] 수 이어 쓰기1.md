# [수학] 수 이어 쓰기1

- 2022-02-11

```python
n = int(input())
leng = len(str(n))
answer = 0

for i in range(leng-1):
  answer += 9 * (10**i) * (i+1)
  
  # 1~9 각 자릿수 길이 : 1 / 자릿수의 합 : 9
  # 10~99 각 자릿수 길이 : 2 / 자릿수의 합 : 90 * 2
  # 100~999 각 자릿수 길이 : 3 / 자릿수의 합 : 900 * 3
  # ex 길이가 15일 때
  # 1~9 까지의 자릿수 합 9 + 나머지는 따로 처리

# (15 - 10 + 1) * 2 = 12 자리
# + 1 해주는 이유 : 10 / 20 / 30 까지 더해줘야함
answer += ((int(n) - (10**(leng-1)) + 1) * leng)

# 따라서 ex 15 의 경우 9 + 12 = 21
print(answer)
```
